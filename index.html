Contéudo abordado durante o curso e o que aprendi:

Durante o curso de HTML básico, aprendi como estruturar uma página de forma semântica e acessível, pensando na experiência de todas as pessoas. Estudamos as principais tags e também como criar formulários com campos como nome, e-mail, data e senha. Vimos a diferença entre as tags antigas do HTML4 e as mudanças que chegaram com o HTML5. Também aprendemos a usar tags de mídia (como áudio e vídeo), criamos tabelas e tivemos um primeiro contato com CSS e JavaScript. Por fim, conhecemos alguns conceitos importantes como Web Scraping, SEO e outros atributos que ajudam no desempenho da página.

Mas o que é um HTML semântico?

"O HTML semântico tem como objetivo descrever o significado do conteúdo presente em documentos HTML, tornando-o mais claro tanto para programadores quanto para browsers e outras engines que processam essa informação." - Dev Media
